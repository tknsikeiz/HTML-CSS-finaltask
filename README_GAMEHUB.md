# GameHub - ゲームコミュニティ・スコア共有

## 🎮 概要
**GameHub** は、ゲーマー向けの総合コミュニティプラットフォームです。  
ゲーム紹介、攻略ガイド、プレイヤーコミュニティ、スコア記録など、ゲーム愛好家向けの多機能サイトです。

---

## 📄 ページ構成（9ページ）

| ページ | ファイル名 | 説明 |
|-------|----------|------|
| ホーム | `index.html` | ゲーム紹介、ランキング、統計情報 |
| ゲーム一覧 | `pet-guide-list.html` | 1000以上のゲーム検索・フィルタリング |
| ゲーム詳細 | `pet-guide-detail.html` | Zelda などゲーム詳細情報 |
| 攻略ガイド | `pet-care-guide.html` | ゲーム別攻略・コツ解説 |
| コミュニティ | `pet-guide-qa.html` | プレイヤーQ&A・情報交換 |
| スコアボード | `pet-guide-record.html` | プレイヤースコア記録・ランキング |
| スコア投稿 | `pet-guide-form.html` | スコア・プレイ記録の投稿フォーム |
